# ♠️ Hearts Card Game

## 📌 Overview
This project is a software implementation of the classic Hearts card game. It is designed using Object-Oriented Programming (OOP) principles to model players, cards, and game logic in a structured and modular way.

## ⚙️ Features
- Standard 52-card deck  
- 1 human player vs 3 AI players  
- Turn-based gameplay  
- Score tracking system  
- Basic AI decision-making 

---

## 🧱 Technologies Used
- Programming Language: Python  
- Concepts: Object-Oriented Programming (OOP), modular design

---

## 📂 Project Structure
- `Card` – Represents a playing card  
- `Deck` – Handles deck creation and shuffling  
- `Player` – Base class for all players  
- `AIPlayer` – Implements computer-controlled logic  
- `Game` – Controls overall game flow 

---

## 🎮 How the Game Works
- Each player is dealt cards  
- Players take turns playing cards  
- Points are calculated based on Hearts rules  
- The player with the lowest score wins
